## ADDED Requirements

### Requirement: 控制預覽音訊播放

系統 SHALL 在預覽來源包含音訊資料時提供可切換的聲音播放控制。

#### Scenario: 預覽來源具有音訊軌

- **WHEN** 系統完成預覽來源建立並判定該來源包含可播放音訊
- **THEN** 前端顯示音訊播放開關
- **AND** 使用者可以切換是否播放該來源的聲音

#### Scenario: 預覽來源不具有音訊軌

- **WHEN** 系統完成預覽來源建立並判定該來源不包含可播放音訊
- **THEN** 前端不播放聲音
- **AND** 前端不顯示或禁用音訊播放開關

#### Scenario: 使用者切換音訊播放狀態

- **WHEN** 使用者在預覽中切換音訊播放開關
- **THEN** 系統更新目前預覽播放器的靜音狀態
- **AND** 不重新建立攝影機來源連線

## MODIFIED Requirements

### Requirement: 啟動即時預覽

系統 SHALL 支援將攝影機 RTSP 串流轉為本機提供的瀏覽器相容影音預覽，並在來源包含音訊時提供同步播放能力。

#### Scenario: 預覽已儲存攝影機

- **WHEN** 使用者已選取一台已儲存攝影機並在主畫面按下預覽
- **THEN** 系統先取得所選碼流類型的 RTSP URI
- **AND** 系統啟動本機預覽工作，產生可供前端播放的新預覽 session 與 URL
- **AND** 系統在新來源可以播放前維持連接中的狀態

#### Scenario: 選擇主碼流或副碼流預覽

- **WHEN** 使用者在主畫面選擇主碼流或副碼流後啟動預覽
- **THEN** 系統依照所選碼流類型取得對應的 RTSP URI
- **AND** 系統使用該來源建立對應的影音預覽

#### Scenario: 預覽來源包含音訊資料

- **WHEN** 系統建立新預覽來源並偵測到來源具有音訊資料
- **THEN** 預覽結果包含可供前端播放音訊的來源資訊

### Requirement: 控制預覽生命週期與前端狀態

系統 SHALL 支援使用者停止預覽，並在切換來源或串流異常時清理舊預覽狀態，只保留目前有效來源的畫面與資訊。

#### Scenario: 使用者手動停止預覽

- **WHEN** 目前存在作用中的預覽且使用者按下停止
- **THEN** 系統關閉目前預覽工作與本機輸出端點
- **AND** 前端清除預覽 URL、串流 URI、音訊狀態與預覽中狀態

#### Scenario: 瀏覽器中的預覽播放失敗

- **WHEN** 前端收到目前預覽播放器的錯誤事件
- **THEN** 前端呼叫停止預覽
- **AND** 系統清除目前預覽狀態

#### Scenario: 切換到新的訊號源

- **WHEN** 使用者切換預覽攝影機或預覽流程開始使用新的訊號源
- **THEN** 前端立即清除舊預覽畫面與舊串流資訊
- **AND** 系統只允許最新訊號源的預覽結果更新畫面
- **AND** 舊訊號源的延遲結果不得重新顯示在畫面上

### Requirement: 於預覽中顯示目前串流 URI

系統 SHALL 在目前有效預覽啟動後提供對應來源的串流 URI 供使用者查看與複製，且切換來源時不得殘留上一個來源的 URI。

#### Scenario: 預覽已啟動且串流 URI 可用

- **WHEN** 目前有效的預覽來源已建立完成且串流 URI 可用
- **THEN** 前端在預覽區顯示目前來源的串流 URI
- **AND** 使用者可將該 URI 複製到剪貼簿

#### Scenario: 預覽切換到新來源

- **WHEN** 預覽流程開始切換到新的攝影機來源
- **THEN** 前端立即清除上一個來源的串流 URI 顯示
- **AND** 只在新來源建立完成後顯示新的串流 URI

### Requirement: 自動預覽已選攝影機

系統 SHALL 支援在使用者切換攝影機選取時自動啟動預覽，並在新來源就緒前避免顯示舊畫面。

#### Scenario: 已啟用自動預覽

- **WHEN** 使用者已開啟自動預覽選項並從已儲存清單選擇某台攝影機
- **THEN** 系統自動取得新來源的串流 URI並開始建立新預覽
- **AND** 前端在新來源可播放前顯示連接中的狀態
- **AND** 前端在新來源就緒後直接顯示新畫面

#### Scenario: 新來源尚未就緒

- **WHEN** 自動預覽已開始切換到新的攝影機但新來源仍在建立中
- **THEN** 前端不得顯示上一台攝影機的預覽畫面
- **AND** 前端不得回退到上一台攝影機的串流資訊

#### Scenario: 未啟用自動預覽

- **WHEN** 使用者未開啟自動預覽選項且切換目前選取的攝影機
- **THEN** 系統停止現有預覽並清除目前畫面
- **AND** 系統等待使用者手動重新啟動預覽
